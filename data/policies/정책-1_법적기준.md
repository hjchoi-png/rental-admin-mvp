# 정책-1. 단기임대 법적 기준 관련 정책

> 매물 유형, 법적 규제, 계약 기간에 관한 기준 정책입니다.

---

## 등록 가능 매물 유형

**SRT 팀 합의 결론**: 오피스텔, 아파트, 빌라, 원룸, 단독주택, 상가주택, 고시원, 펜션, 쉐어하우스, 게스트하우스, 생활형숙박시설

### 유형별 분류표

| 분류 | 등록 가능 매물 |
|------|--------------|
| 오피스텔 | 오피스텔 |
| 아파트 | 아파트 |
| 일반 주택 | 원룸, 연립·빌라, 상가주택, 단독주택 |
| 특수 | 생활형숙박시설, 게스트하우스, 고시원, 쉐어하우스 |
| **등록 불가** | **호텔, 모텔/여관, 펜션(별도 허가 없는 경우)** |

> 호텔/모텔은 이용자 인식상 100% '숙박 대상 매물'이므로 제외. 서비스 정체성(임대차) 유지 목적.

---

## 매물 유형별 등록 안내 가이드

### 건물 유형 선택 기준

| 건물 유형 | 선택 기준 |
|----------|---------|
| 오피스텔 | 업무시설로 등록된 오피스텔 포함 |
| 아파트 | 공동주택(아파트) 호실을 등록하는 경우 선택 |
| 빌라 | 연립·다세대 등 저층 공동주택(빌라)에 해당하는 경우 선택 |
| 원룸 | 다가구 건물 중 소형 주거 형태(원룸 구조 기준)로 등록할 때 선택 |
| 단독주택 | 한 세대가 단독으로 사용하는 단독주택/다가구(한 가구 기준) |
| 상가주택 | 상가 + 주택이 함께 있는 건물인 경우 선택 |
| 고시원 | 고시원 사업자로 등록된 경우에만 선택 (고시텔/원룸텔 포함) |
| 쉐어하우스 | 공용 공간과 독립된 개인 공간이 모두 있는 형태 |
| 게스트하우스 | 숙박업에 해당하며, 게스트하우스 사업자 등록된 경우에만 선택 |
| 생활형숙박시설 | 생활형 숙박시설(생숙) 용도의 건물인 경우 선택. 숙박업 허가 필수 |

### 특수 유형 등록 시 체크박스 확인 필요

등록 시 아래 카테고리 선택 시 호스트에게 동의 확인 문구 표시:

**고시원 선택 시:**
☐ 고시원 사업자 등록된 경우에만 해당하며, 관련 사업자 등록 및 허가 요건을 충족합니다.

**게스트하우스 선택 시:**
☐ 게스트하우스는 숙박업에 해당하며, 관련 사업자 등록 및 허가 요건을 충족합니다.

**생활형숙박시설 선택 시:**
☐ 생활형숙박시설은 숙박업 허가를 받은 경우에만 등록 가능하며, 관련 사업자 등록 및 허가 요건을 충족합니다.

> 사업자등록증 등 추가 서류는 받지 않음 (등록 허들 최소화 목적)

### 다인실 제한
- 쉐어하우스, 게스트하우스 등 다인실(도미토리 등)은 등록 불가
- **1인실만 등록 가능**
- 노출 문구: "쉐어하우스는 1인실만 등록 가능합니다." / "게스트하우스는 1인실만 등록 가능합니다."

## 계약(예약) 기간 정책

**SRT 팀 합의 결론**: 최소 1주(7일) ~ 최대 12주(약 3개월) / 또는 1개월~3개월

- 직방 원오빌(월세) 사업과의 충돌 우려로 6개월 이상 제외
- 법무팀 검토 결과, 단기임대 안전 구간은 3개월(90일)까지가 합리적

### 법적 근거

주택임대차보호법 제11조:
> "이 법은 일시사용하기 위한 임대차임이 명백한 경우에는 적용하지 아니한다."

다수 판례 기준, 다음을 충족 시 단기임대차로 볼 확률이 큼:
1. 무보증금 또는 소액보증금
2. 6개월 이하의 임대차 기간
3. 차임 선지급 조건

### 계약 기간 구조

| 구분 | 기간 |
|------|------|
| 최소 계약 기간 | 7일 (1주) |
| 최대 계약 기간 | 90일 (3개월, 약 12주) |
| 일 단위 추가 | 최소 7일 충족 후 1일 단위 추가 가능 |

**예시**: 7일(필수) + 2일 추가 = 9일 계약 가능

---

## 일단위 연장 계약 가격 산식

**SRT 팀 합의 결론**: (임대료 + 관리비) ÷ 7

- 청소비는 1회성 비용이므로 산식에서 제외
- 보증금은 반환형이므로 산식에서 제외

### 기간 정책 상세

| 구분 | 내용 |
|------|------|
| 최초 예약 최소 기간 | 1주(7일) |
| 신규(연장) 계약 최소 기간 | 1주(7일) |
| 일 단위 추가 | 최소 7일 + 1일 이상 추가 가능 |

> 연장 계약은 새로운 계약(신규 계약)으로 처리됨. 자동 연장 없음.

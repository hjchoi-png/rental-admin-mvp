# 정책-2. 단기임대 가격 정책

> 플랫폼 수수료, 가격 구성, 할인 정책, 환불/위약금, 진행 프로세스, 분쟁 처리에 관한 기준 정책입니다.

**핵심 수수료율**: 호스트 3%, 게스트 9% (부가세 별도) | 보증금 30만원 고정

---

## 플랫폼 수수료

단기임대 서비스는 안전한 거래 중개와 시스템 운영을 위해 호스트와 게스트에게 각각 플랫폼 이용 수수료를 부과합니다. 수수료는 계약 체결 시점에 자동으로 계산되며, 결제 화면에서 명확히 확인할 수 있습니다.

**확정된 수수료율**: 호스트 3%, 게스트 9% (부가세 별도)

| 구분 | 수수료율 | 부과 기준 |
|------|---------|---------|
| 호스트 (임대인) | 3% | (임대료 + 관리비 + 청소비) × 3% |
| 게스트 (임차인) | 9% | (임대료 + 관리비 + 청소비) × 9% |

**수수료 부과 기준:**
- 임대료, 관리비, 청소비 합계액에 대해 수수료 부과
- 보증금은 반환형 금액이므로 수수료 부과 대상에서 **제외**
- 할인이 적용된 경우, 할인 후 금액을 기준으로 수수료 계산

---

## 신규 서비스 수수료 할인 [검토중]

초기 호스트 공급 확보 목적으로 신규 호스트 대상 한시적 수수료 할인 정책 운영 예정.

| 옵션 | 할인 기간 | 적용 방식 | 근거 |
|------|---------|---------|------|
| A. 첫 6개월 0% | 6개월 | 가입 후 자동 적용 | 초기 2~3개월 문의·거래 공백 보상 |
| B. 첫 3개월 0% | 3개월 | 가입 후 자동 적용 | 첫 계약 평균 주기(10~12주) 부합. 개발 리소스 최소화 |
| C. 첫 10건 0% | 거래 건수 기준 | 예약 체결 시점 | 성과 기반 인센티브 |

---

## 가격 구성 (임대료 / 관리비 / 청소비 / 보증금)

**SRT 팀 합의 결론**: 보증금 30만원 고정. 가격 화면에 임대료·관리비·청소비·보증금 따로 표시 + 총합 표시.

| 항목 | 정의 | 과금 기준 | 수수료 부과 |
|------|------|---------|-----------|
| ① 임대료 | 공간 사용에 대한 기본 비용 | 박/주 단위 | ✅ (할인율 적용 대상) |
| ② 관리비 | 공과금(수도·가스·전기) 및 인터넷·건물 관리비 포함 | 박/주 단위 | ✅ |
| ③ 청소비 | 입실 전(퇴실 후) 1회 수행되는 청소 및 세탁 비용 | 건 단위 (1회) | ✅ (연장 시 추가 부과 없음) |
| ④ 보증금 | 기물 파손·오염·실내 흡연 등에 대비한 예치금 | 건 단위 (1회) | ❌ (반환형, 수수료 제외) |

## 할인 정책

### 장기 계약 할인

**SRT 팀 합의 결론**: 호스트가 주(Week) 단위 구간별 할인율(%)을 직접 입력. 할인율은 **임대료에만** 적용.

- 기본 설정 구간: 2주 / 3주 / 4주 / 8주 이상
- 구간 추가 가능 (기능으로 구현)

| 계약 기간 | 할인율 |
|---------|------|
| 2주 | 호스트 설정 (예시: 5%) |
| 3주 | 호스트 설정 (예시: 7%) |
| 4주 | 호스트 설정 (예시: 10%) |
| 8주 이상 | 호스트 설정 (예시: 15%) |

> 호스트에게 최적 할인율 가이드 제안 후 최종 결정권은 호스트에게 부여

### 즉시입주 할인

**SRT 팀 합의 결론**: 호스트가 직접 결정. 할인율은 **임대료에만** 적용. 정액(Fixed Amount) 차감 방식.

- 입주일이 임박한 매물에 제공하는 정액 할인 (1만원, 2만원 단위)
- 공실률 최소화 및 호스트 가격 통제권 강화 목적
- 퍼센트(%) 방식 대신 정액 방식 채택 → 단기 체류 게스트에게 심리적으로 더 직관적

### 성수기/비수기 차등 가격 [검토중]

특정 기간(날짜 범위)을 지정하여 기본 요금과 다른 특별 요금(증액 또는 할인)을 적용.
- 할인율은 **임대료에만** 적용
- 성수기/비수기 모두 구간별 증액율/할인율 적용 방식 (구간 추가 가능)

## 환불 및 위약금 정책

**SRT 팀 합의 결론**: 유연(Flexible) / 표준(Standard) / 엄격(Strict) 3단계 위약금표

### 위약금표

| 취소 시점 | 유연 (Flexible) | 표준 (Standard) | 엄격 (Strict) |
|---------|--------------|--------------|------------|
| 입주 14일 전 | 무료 취소 | 임대료의 10% | 임대료의 20% |
| 입주 7일 전 | 임대료의 15% | 임대료의 30% | 임대료의 50% |
| 입주 3일 전 | 임대료의 30% | 임대료의 40% | 임대료의 60% |
| 입주 1일 전 | 임대료의 40% | 임대료의 50% | 임대료의 70% |
| 입주 당일 | 취소 불가 | 취소 불가 | 취소 불가 |

| 위약금 단계 | 추천 대상 |
|-----------|---------|
| 유연 (Flexible) | 공실을 빨리 채우고 싶은 매물 |
| 표준 (Standard) | 일반적인 단기임대 매물 |
| 엄격 (Strict) | 성수기/고가 매물 |

### 위약금 공통 규정

- 단기임대는 부동산 임대차 계약의 한 형태입니다.
- 계약 취소 시 취소 사유가 있는 당사자에게 고지된 환불 규정에 따라 위약금이 발생합니다.
- **입주 당일 및 거주 중 취소는 불가**합니다.
- 계약은 임차인(게스트)의 결제와 동시에 체결됩니다.
- 임대료는 결제(예약) 시점을 기준으로 합니다.
- **위약금은 임대료에만 적용**되며, 관리비/청소비/보증금은 전액 환불됩니다.
- 취소 가능 기간 및 위약금 발생 시점은 입주 예정일 00:00를 기준으로 역산하여 계산합니다.

### 당일 결제 취소 특례

- 결제 당일 취소 시: 환불 규정과 무관하게 **임대료의 10% 위약금** 적용 (당일 24:00 기준)
- 단, 무료 취소 기간에 해당하면 전액 환불

### 계약 수수료 정책 (취소 시)

| 취소 사유 | 게스트 수수료 | 호스트 수수료 |
|---------|------------|------------|
| 무료 취소 | 없음 | 없음 |
| 게스트 사유 취소 | 위약금 기준 수수료율 부과 | 위약금 기준 수수료율 부과 |
| 호스트 사유 취소 (입주 전) | 없음 | 없음 |
| 호스트 사유 취소 (거주 중) | 수수료 환불 불가 | 수수료 환불 불가 |

### 환불 규정 — 게스트

- 위약금표에 따라 적용된 금액과 계약수수료를 공제한 나머지 금액만 환불
- 입주 당일 취소: 임대료 환불 없음, 관리비/청소비/보증금은 환불
- 거주 중 퇴실: 환불 없음, 퇴실 확인 절차로 동일 프로세스 진행

### 환불 규정 — 호스트

- 확정된 단기임대 계약 취소는 게스트와 상호 협의 후 진행 가능
- 입주 전 호스트 사유 취소 시 → 선택된 환불 규정에 따라 게스트에게 위약금 지급
- 거주 중 호스트 사유 취소는 기본적으로 불가 (호스트-게스트 상호 협의 필요)

## 진행 프로세스 및 기한

| 단계 | 처리 주체 | 기한 | 미응답 시 |
|------|---------|------|---------|
| 예약 요청 | 게스트 → 호스트 | — | — |
| 예약 승인 | 호스트 | **24시간 이내** | 자동 예약 거절 |
| 결제 | 게스트 | 승인 후 **6시간 이내** | 자동 취소 |
| 계약 체결 | 자동 | 결제 완료 시 | — |
| 임대료 정산 | 플랫폼 → 호스트 | 게스트 입주 확인 후 **영업일 2일 이내** | 입주일 기준 영업일 10일 내 자동 정산 |
| 퇴실 요청 | 게스트 | 퇴실 시간 기준 | **24시간 내 미응답 시 자동 퇴실 처리** |
| 퇴실 확인 (점검) | 호스트 | 게스트 퇴실 요청 후 | **24시간 내 미응답 시 자동 퇴실 확인 처리** |
| 보증금 보류 | 호스트 | 퇴실 확인 단계 | 최대 10일 (영업일·비영업일 포함) |
| 보증금 반환 | 플랫폼 → 게스트 | 퇴실 확인 후 **영업일 2일 이내** | 최대 퇴실일로부터 10일 |

> 계좌이체 결제 시 영업일 2일 이내 반환. 카드 결제 시 카드사에 따라 최대 7일 소요.

## 분쟁 처리

**대전제**: 플랫폼은 **기록자·자동처리자**. 개별 계약 분쟁에 개입하지 않음. 중재·협의 요청·판단·강제 집행 수행하지 않음.

### 호스트 계약 취소 처리 흐름

```
호스트 취소 요청 (사유 + 위약금 입금)
  → 자동 처리
  → 게스트에게 자동 안내 메시지
  → 게스트 '취소 수락' 동의
  → 게스트에게 입금 (플랫폼 보유 금액 + 호스트 위약금)
  → 호스트 "취소 완료" 메시지 전달
```

> 호스트가 위약금을 입금한 후에만 취소 처리 진행 가능 (미입금 시 취소 불가)

### 보증금 보류 및 차감 처리 흐름

```
호스트 '보증금 보류 하기'
  → 호스트가 합의 내용 및 증빙 자료 플랫폼에 제출
  → 자동 처리
  → 게스트 '내용 수락' 동의
  → 차감 후 잔여 보증금 게스트에게 반환
  → 호스트 "처리 완료" 메시지 전달
```

### 보증금 차감 인정 사유

| 사유 | 정의 | 예시 |
|------|------|------|
| 시설 파손 (고정 시설물) | 벽면·바닥·창호 등 고정 마감재의 물리적 훼손 | 벽지 찢김·변색, 바닥 찍힘·긁힘, 유리창 파손, 도어락 파손 등 |
| 물품 파손·분실 (제공 옵션) | 호스트가 제공한 가전·가구·생활 집기의 고장·파손·분실 | TV 액정 파손, 리모컨 분실, 소파/매트리스 심한 얼룩 등 |
| 관리비 초과 (공과금) | 계약 시 약정된 기본 공과금 지원액을 초과하여 발생한 요금 | 동절기 보일러 과다 사용, 에어컨 미소등으로 인한 기준액 초과분 |

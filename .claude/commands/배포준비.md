# 배포 준비 워크플로우

프로덕션 배포 전 최종 체크리스트를 실행하라.
(finishing-a-development-branch + verification-before-completion 스킬)

## 단계

### 1단계: 빌드 검증
- `npx next build` 실행
- TypeScript 에러 0건 확인
- 빌드 경고 목록 확인 및 보고

### 2단계: 환경 변수 확인 (secrets-management 스킬)
- `.env.local`에 필요한 모든 변수가 있는지 확인
- 프로덕션 환경에 필요한 변수 목록 제시
- 시크릿이 코드에 노출되어 있지 않은지 검사

### 3단계: DB 마이그레이션 확인 (database-migration 스킬)
- 적용 대기 중인 마이그레이션이 있는지 확인
- 마이그레이션 파일의 안전성 검토 (롤백 가능 여부)

### 4단계: 테스트 실행
- 유닛 테스트 전체 실행 (있는 경우)
- 결과 보고

### 5단계: 변경사항 요약
- 최근 커밋 이후 변경사항 정리
- 배포 시 주의사항 정리
- 배포 체크리스트 제공

## 출력
배포 가능 여부를 명확히 판정하고, 블로커가 있으면 해결 방법 제시.

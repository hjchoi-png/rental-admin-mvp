# 마무리 커맨드

세션을 마무리하고 회고를 자동으로 기록합니다.

## 실행 순서

### 1. 회고 스킬 자동 실행
- `/회고` 스킬을 자동으로 실행
- 이번 세션에서 완료한 작업 요약
- 빌드 상태 확인 (TypeScript, npm build)
- 잘된 점, 아쉬운 점, 다음 할 일 정리

### 2. progress.md 자동 업데이트
- docs/progress.md에 세션 기록 자동 추가
- **사용자 확인 없이** 바로 추가 (자동화)
- 최신 세션이 맨 위로 오도록 정렬

### 3. 완료 안내
```
✅ Session N 기록이 추가되었습니다!

다음 세션 시작 시:
"CLAUDE.md + progress.md 최신 세션 읽어줘"
```

## 사용 시점

- 세션 종료 시
- 작업이 일단락되었을 때
- 다음 세션으로 넘기기 전

## 자동화 정책

- **회고 생성**: 자동
- **progress.md 업데이트**: 자동 (확인 안 함)
- **빌드 검증**: 자동 (에러 있어도 기록)

---

**참고:** 이 커맨드는 회고 스킬을 자동으로 호출하므로, 별도로 `/회고` 를 실행할 필요가 없습니다.

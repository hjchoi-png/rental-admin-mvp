## 스킬 작동 방식

1. **자동 활성화**: 관련 코드 작업 시 에이전트가 해당 스킬의 SKILL.md를 자동으로 로드
2. **글로브 매칭**: 파일 패턴(예: `*.tsx`, `*.sql`)에 따라 관련 스킬 활성화
3. **슬래시 커맨드**: `/pdf`, `/docx` 등으로 수동 호출 가능
4. **설치 위치**: `.agents/skills/` (universal) + `.claude/skills/` (Claude Code 심링크)
5. **권한**: 에이전트 권한으로 실행되므로 설치 전 SKILL.md 내용 검토 권장

## 1. 문서 목적

단기임대(STR) 매물 관리 플랫폼 **어드민 시스템**의 현황 분석과 **AI 기반 자동화·운영 효율화** 개선 방향을 제시합니다.  
매물 검수, 데이터 품질, 운영 효율을 위한 **기능 명세**와 **단계별 로드맵**을 포함합니다.

---

## 2. 현재 시스템 (rental-admin-mvp 기준)

| 구분 | 내용 |
|------|------|
| **기술 스택** | Next.js 14, React 18, TypeScript, Supabase, OpenAI GPT-4o, Radix UI, TailwindCSS, shadcn/ui, TanStack Table |
| **구현된 기능** | 매물 6단계 등록 폼, 이미지 업로드(HEIC 변환), 어드민 대시보드, 매물 목록·상태별 필터, 수동 승인/반려, AI 매물 검수(5항목 100점), Daum 주소 API |
| **한계** | AI 검수 수동 트리거, 완전 수동 승인, 일괄 처리 없음, 운영 통계·알림·매물 비교 없음 |

---

## 3. 핵심 개선 방향 (요약)

1. **AI 기반 자동화**: 등록 즉시 자동 AI 검수, 임계값 기반 자동 승인, 이미지/텍스트 자동 개선, AI 가격 추천  
2. **운영 효율화**: 일괄 승인/반려, 고도화된 필터·검색, 반려 사유 템플릿, 감사 로그  
3. **데이터 인사이트**: 일/주/월 통계 대시보드, 품질 트렌드, 지역·유형별 가격 분석  

---

## 4. 로드맵 타임라인 (14주)

| Phase | 기간 | 주요 내용 | 목표 |
|-------|------|-----------|------|
| **Phase 1** | 4주 | 자동 AI 검수, 자동 승인, 일괄 처리, 반려 템플릿 | 수작업 50% 감소 |
| **Phase 2** | 4주 | 이미지 최적화, 텍스트 교정, AI 가격 추천 | 매물 품질 향상 |
| **Phase 3** | 3주 | 알림(이메일/시스템), 운영 대시보드, 감사 로그 | 데이터 기반 의사결정 |
| **Phase 4** | 3주 | 고급 검색, 매물 비교, 성능·모바일 최적화 | 완성도·UX 향상 |

**총 14주 (약 3.5개월)**

---

## 5. 기대 효과 요약

- **운영**: 검수 시간 70% 단축, 일일 처리 매물 수 3배  
- **품질**: 일관된 기준, 저품질 사전 차단, 이미지/텍스트 자동 개선  
- **KPI**: 검수 시간 30분→5분 이하, 일일 처리 50→150개, 평균 AI 점수 60→75점 이상
